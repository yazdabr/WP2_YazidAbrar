Beranda Tentang Layanan Kontak

Artikel 1 - HTML

HTML (HyperText Markup Language): Bahasa markup yang digunakan untuk membuat struktur halaman web dengan elemen-elemen seperti heading, paragraf, gambar, dan link.

Artikel 2 - CSS

CSS (Cascading Style Sheets): Bahasa yang digunakan untuk mengatur tampilan dan gaya elemen-elemen HTML, seperti warna, font, dan layout.

Artikel 3 - JavaScript

JavaScript: Bahasa pemrograman yang digunakan untuk menambahkan interaktivitas dan dinamika pada halaman web, seperti efek animasi, validasi formulir, dan interaksi pengguna.
